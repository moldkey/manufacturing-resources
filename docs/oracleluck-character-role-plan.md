## 一、背景

### 1.1 为什么做

Character.AI 月访问 1.6 亿，用户平均聊 14 分钟。核心逻辑：**AI + 角色 = 关系**。用户不是来用工具的，是来和角色对话的。

OracleLuck 有 1M+ 页的古籍/诗词/玄学内容，天然适合做"角色对话"——用户找诗人聊诗、找哲学家答疑、找神仙排解烦恼。这和 Character.AI 的底层结构一致，但赛道不同（文化/玄学 vs 通用娱乐）。

### 1.2 定位

```
OracleLuck 角色对话 = 传统文化 × AI 人格化体验
不是算命（不做预测）
不是通神（不装神弄鬼）
是「角色化的文化对话体验」
```

### 1.3 与 Character.AI 的关键区别

| 维度 | Character.AI | OracleLuck |
|:----|:------------|:----------|
| 角色类型 | 任何角色（动漫/游戏/名人/虚构） | 仅限传统文化角色（诗人/哲学家/神仙/占卜） |
| 用户创建角色 | 全部开放 | 有限开放（不可创建真人/宗教偶像名） |
| 内容审核 | 2024年后才加强 | **从一开始做** |
| 服务器 | 美国 | **香港** — 合规空间更大 |
| 内容资产 | 无自有内容 | **1M 页古籍/诗词/玄学内容** |
| SEO | 弱（靠用户分享） | **强（每角色=一页SEO，内链到古籍内容）** |

## 二、合规边界（红线）

服务器在香港，香港法律对命理/占卜/神佛对话无专门禁令。但面向全球华人用户（含大陆），且用户要求"不能过份"。

### 🔴 红线清单（绝对不做）

```
1. ❌ 不扮演真人（三甲医院医生、心理咨询师、律师等）
2. ❌ 不对未成年人开放神仙/占卜/命理类角色
3. ❌ 不接受用户对"神仙"打赏/送礼物（避免宗教敛财嫌疑）
4. ❌ 不做出预测性结论（"你七月会发财""你明年有灾"）
5. ❌ 不伪造"通灵/显灵/神迹"体验
```

### 🟡 底线规则（对话内容）

```
- 每条回复显式标注「AI生成 | 传统文化体验」
- 用户输入必须过审核（敏感词/危机词检测）
- 严格禁止回复：
  - 医疗/法律/财务建议
  - 任何"一定会/一定不会"句式
  - 自残/自杀/伤害他人——必须弹救助热线
- 对话记录按法规留存
```

### 3.3 角色数据结构

```sql
-- 已在 04-神仙对话产品设计.md 中设计，需要微调的表结构：

CREATE TABLE characters (
  id BIGSERIAL PRIMARY KEY,
  slug TEXT UNIQUE NOT NULL,
  name TEXT NOT NULL,             -- 显示名
  title TEXT,                     -- 称号
  type TEXT NOT NULL,             -- poet/philosopher/deity/wisdom
  icon TEXT,                      -- emoji
  image_url TEXT,                 -- 头像
  personality TEXT,               -- 人设描述（用于 prompt）
  speaking_style TEXT,            -- 说话风格
  domains TEXT[],                 -- 擅长领域
  system_prompt TEXT NOT NULL,    -- AI 人设提示词
  knowledge_base TEXT[],          -- 关联的古籍/知识源 URL 列表
  compliance_level TEXT NOT NULL DEFAULT 'culture',  -- culture/light_wisdom
  is_premium BOOLEAN DEFAULT FALSE,
  is_active BOOLEAN DEFAULT TRUE,
  sort_order INTEGER DEFAULT 0,
  created_at TIMESTAMPTZ DEFAULT NOW()
);
```

### 3.4 角色提示词设计原则

每条回复必须符合"角色人设"：
- **诗人**：引用自己的诗词，用诗人身份谈文学、人生、自然
- **哲学家**：引用经典，字斟句酌地解答
- **神仙**：用神佛口吻安慰开导，但不做预测
- **文化导师**：引用古籍知识做文化科普，不评判未来

示例回复边界（关公版）：

```
用户问："关公，我今年能发财吗？"
❌ 错误："你今年财运不错，八月会有转机"
✅ 正确："关某在京剧中常说'义字当头'。赵公明那老头儿说了，
    得财先讲德。你且说说你做的什么营生，关某帮你参详。"
```

  3. AI 互动小说（剧情推进）

### 6.3 关键风险

- **封建迷信风险**：不预测、不通神、不打赏，守住红线
- **未成年人保护**：P1 就要做年龄验证，不拖
- **AI 内容质量**：角色 prompt 需要持续调优，不是一版搞定
- **用户量**：冷启动阶段角色太少可能不够吸引用户
